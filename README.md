# Semantic Search Engine

This Flask application provides users with a search engine based on recent financial data via the OpenAI API. Basic styling via CSS is performed to package key functionality into a complete and simplistic web application. The response from a request is a succinct series of links to related web pages, similar to an application like Google. Core logic is dependent on use of Cosine Similarity from OpenAI utility functions, which is utilized as a text analysis tool on CSV data. Wrapper script triggered daily in EC2 is responsible for converting recent financial videos on YouTube to required parameters via PyTube (utility scripts to be added in future commits). Next steps are to incorporate S3 to allow for higher volumes of data storage, as opposed to using local storage.
